About This Project

Understanding our NEPSE data archive initiative

info Project Overview

What We Do

This project archives historical NEPSE (Nepal Stock Exchange) data from ShareSansar, making it easily accessible for:

  • Market analysis and research
  • Historical trend analysis
  • Data-driven investment decisions
  • Academic and educational purposes

Data Source

All data is sourced from ShareSansar, a leading financial news portal in Nepal. This is an unofficial archive project and is not affiliated with ShareSansar or NEPSE.

Updates & Maintenance

The data is updated regularly to ensure you have access to the latest market information. However, please note:

  • This is an unofficial archive
  • Data accuracy depends on the source
  • Use the data at your own discretion
